Detailed Description

template<ObjectType T>
struct dbs_map::model::simplified::Lines< T >

Data structure representing line geometry for a specific type of object.

This template structure defines a Lines object which represents the line geometry of a specific object type in the railway system. Each Lines object is defined by its unique identifier (ID), the parent object's ID, and a set of 3D polylines.
