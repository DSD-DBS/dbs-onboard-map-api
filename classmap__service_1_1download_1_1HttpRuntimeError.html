Detailed Description

Custom exception class for handling HTTP errors.

This class extends from std::runtime_error and is thrown when an HTTP error occurs. Specifically, it is thrown when the HTTP status code is 300 or greater. It includes methods for providing the reason and HTTP status code of the error.
