Detailed Description

Provides network access to a specific map layer in a DB map catalog.

This class is used to interact with a specific layer of a DB map catalog over the network. It allows retrieving metadata for the layer, the partition metadata for a specific partition ID, retrieving partition content via data handle , getting the differences between two versions of the layer and retrieving all partitions metadata.
