Detailed Description

Data structure representing an edge in a railway track graph.

This structure defines a TrackEdge object which represents a specific edge or connection between nodes in a railway track graph. Each edge is defined by its unique identifier (ID), the nodes it connects (side_a and side_b), and its spatial geometry.
