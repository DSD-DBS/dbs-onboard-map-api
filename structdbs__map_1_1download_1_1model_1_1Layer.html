Detailed Description

Data Transfer Object (DTO) representing metadata for a layer in a map catalog.

This structure is used to transfer metadata about a specific layer within a map catalog across different parts of the system or network. The metadata includes the IDs of the catalog and layer, zoom levels, schema name, partitioning type, content type and encoding, and description of the layer.
