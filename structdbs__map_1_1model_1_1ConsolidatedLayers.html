Detailed Description

Data structure representing a consolidated view of all map layers.

This structure defines a ConsolidatedLayers object which provides a consolidated view of all the map layers in the database. This includes the topology of the railway system, 3D landmarks, risk assessment zones, and the associated data partitions (tiles) representing specific areas of the railway system. Each ConsolidatedLayers object also maintains a list of PartitionIds which are used for efficient data management and querying.
