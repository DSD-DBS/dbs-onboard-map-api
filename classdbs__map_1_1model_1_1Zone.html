Detailed Description

Class representing a specific zone for risk assessment.

This class defines a Zone object which represents an area in which a specific risk assessment is done. These zones can be of different types such as RiskZone, SafeZone, LevelCrossingZone, etc. Each zone is defined by its type, unique identifier (ID), and 3D geometry.
