Detailed Description

This class serves as the main interface for accessing and updating onboard map data.

It provides functionality to retrieve map data in WSG84 coordinates, which can be requested by various geographical areas: geo rectangle, corridor or by tile IDs. It also offers Over The Air (OTA) update functionality for map data.
